# C++ Array Statistics System

A menu-driven C++ program that stores and analyzes numeric data using arrays and sorting algorithms.

## Features
- Insert numbers into an array
- Display largest value
- Display smallest value
- Calculate median
- Calculate average
- Sort/display numbers in ascending order

## Concepts Demonstrated
- Arrays
- Functions
- Bubble Sort
- Insertion Logic
- Loops and Conditionals
- Menu-Driven Programming

## Technologies
- C++
- Console-based interface
